The UartRemote class¶

class uartremote.UartRemote(port=0, baudrate=115200, timeout=1500, debug=False, rx_pin=18, tx_pin=19)¶

Bases: object

This is a class used to communicate via REPL or some kind of RPC command loop with other devices over a UART interface.

Parameters

  • port – The hardware uart port used to by UartRemote class

  • baudrate – The baudraste at which the UART communicates, defaults to 115200

  • timeout – The timeout (in ms) for waiting for data coming in on the UART, default to 1500

  • debug – A flag to generate debug output, default to False

  • rx_pin – The pin on which the UART will receive information, default to 18

  • tx_pin – The pin on which the UART will receive information, default to 19
